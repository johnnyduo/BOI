## 3. รูปแบบโฆษณา YouTube

### 3.1 Skippable In-stream Ads (แนะนำ)
```
- ความยาว: ไม่จำกัด (แนะนำ 15-60 วินาที)
- ข้ามได้หลัง: 5 วินาที
- จ่ายเมื่อ: ดูครบ 30 วิ หรือ คลิก
- เหมาะกับ: Brand Awareness, Video Views
- CPV เฉลี่ย: 0.30-0.80 บาท
```

### 3.2 Non-skippable In-stream Ads
```
- ความยาว: 15-20 วินาที (max)
- ข้ามไม่ได้
- จ่ายเมื่อ: ต่อ 1,000 impressions (CPM)
- เหมาะกับ: Full message delivery
- CPM เฉลี่ย: 80-150 บาท
```

### 3.3 Bumper Ads
```
- ความยาว: 6 วินาที (max)
- ข้ามไม่ได้
- จ่ายเมื่อ: CPM
- เหมาะกับ: Remarketing, Brand recall
- CPM เฉลี่ย: 40-80 บาท
```

### 3.4 In-feed Video Ads (Discovery)
```
- แสดงที่: หน้า Search, หน้า Home, Related videos
- Thumbnail + Title + Description
- จ่ายเมื่อ: คลิกดูวิดีโอ
- เหมาะกับ: Subscriber Growth, Engagement
- CPV เฉลี่ย: 1-3 บาท
```

### 3.5 YouTube Shorts Ads (ใหม่ & ทรงพลัง!)
```
- ความยาว: 10-60 วินาที (แนวตั้ง)
- แสดงระหว่าง Shorts feed
- จ่ายเมื่อ: CPV หรือ CPM
- เหมาะกับ: Gen Z, Viral content
- CPV เฉลี่ย: 0.10-0.30 บาท (ถูกมาก!)
```

## 5. Video Specs & Requirements

### In-stream Ads
| รายการ | สเปค |
|--------|------|
| Resolution | 1920 x 1080 (1080p) หรือ 1280 x 720 (720p) |
| Aspect Ratio | 16:9 |
| Format | .MP4, .MOV, .AVI |
| File Size | Max 1GB (แนะนำ < 500MB) |
| Frame Rate | 24, 25, หรือ 30 fps |
| Audio | AAC-LC, 128kbps+ |

### YouTube Shorts Ads
| รายการ | สเปค |
|--------|------|
| Resolution | 1080 x 1920 (9:16) |
| Aspect Ratio | 9:16 (แนวตั้ง) |
| Length | 10-60 วินาที |
| Format | .MP4 |

### Thumbnail
| รายการ | สเปค |
|--------|------|
| Resolution | 1280 x 720 px |
| Format | .JPG, .PNG |
| File Size | Max 2MB |
| Text | ไม่เกิน 20% ของภาพ |
