## 6. Budget Reallocation Rules

### เมื่อไหร่ควรเพิ่มงบ

```
✅ เพิ่มงบเมื่อ:
├── CTR > 3% (ดีกว่าค่าเฉลี่ย 50%)
├── View Rate > 35%
├── CPA < target 20%
├── ROAS > 1.5x
└── Subscriber Cost < 25 บาท
```

### เมื่อไหร่ควรลดงบ

```
⚠️ ลดงบเมื่อ:
├── CTR < 1%
├── View Rate < 15%
├── CPA > target 50%
├── Frequency > 5/week
└── Subscriber Cost > 50 บาท
```

### เมื่อไหร่ควรปิดแคมเปญ

```
❌ ปิดเมื่อ:
├── CTR < 0.5% หลัง 7 วัน
├── ไม่มี Conversions หลัง 14 วัน
├── CPA > 2x target
└── Budget spent แต่ผลเป็น 0
```
