## 7. ROI Calculation

### สูตร ROAS
```
ROAS = รายได้จากโฆษณา / งบโฆษณา

ตัวอย่าง:
งบโฆษณา = 100,000 บาท
รายได้ = 250,000 บาท
ROAS = 250,000 / 100,000 = 2.5x
```

### ROI Tracking Table
| เดือน | งบโฆษณา | รายได้ | ROAS | หมายเหตุ |
|-------|---------|--------|------|---------|
| เดือน 1 | | | x | |
| เดือน 2 | | | x | |
| เดือน 3 | | | x | |
| รวม | 100,000 | | x | |

### Customer Lifetime Value (CLV)
```
CLV = ค่าเฉลี่ยต่อการซื้อ × จำนวนครั้งที่ซื้อต่อปี × จำนวนปี

ตัวอย่าง:
ค่าเฉลี่ยต่อคอร์ส = 3,000 บาท
ซื้อ 2 คอร์ส/ปี
อยู่กับแบรนด์ 3 ปี
CLV = 3,000 × 2 × 3 = 18,000 บาท
```
