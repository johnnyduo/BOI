## 9. Content Ideas by Week

### Week 1-2: Foundation
```
หัวข้อหลัก: "ทำไมต้องลงทุน?"

Content Ideas:
1. "เงินเฟ้อกินเงินออมคุณอย่างไร"
2. "พลังของดอกเบี้ยทบต้น"
3. "เริ่มลงทุนตอนอายุเท่าไหร่ดี?"
4. "ลงทุน vs เก็บเงิน ต่างกันอย่างไร"
5. "5 ข้อผิดพลาดที่มือใหม่ทำ"
```

### Week 3-4: Basic Investment
```
หัวข้อหลัก: "เริ่มต้นลงทุน"

Content Ideas:
1. "หุ้น กองทุน คริปโต เลือกอะไรดี?"
2. "เงินเท่าไหร่ถึงลงทุนได้?"
3. "SET Index คืออะไร?"
4. "อ่านงบการเงินเบื้องต้น"
5. "เทคนิค DCA คืออะไร?"
```

### Week 5-6: Intermediate
```
หัวข้อหลัก: "ลงทุนอย่างชาญฉลาด"

Content Ideas:
1. "Asset Allocation คืออะไร?"
2. "กระจายความเสี่ยงอย่างไร?"
3. "เมื่อไหร่ควรขาย?"
4. "Technical vs Fundamental"
5. "เลือกโบรกเกอร์อย่างไร?"
```

### Week 7-8: Advanced
```
หัวข้อหลัก: "สร้างความมั่งคั่ง"

Content Ideas:
1. "วางแผนเกษียณ"
2. "ภาษีกับการลงทุน"
3. "สร้างพอร์ตระยะยาว"
4. "เตรียมตัวสำหรับวิกฤต"
5. "สรุป: เส้นทางสู่ความมั่งคั่ง"
```
