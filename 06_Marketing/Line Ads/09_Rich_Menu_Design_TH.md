## 1. Rich Menu คืออะไร?

Rich Menu คือเมนูที่แสดงอยู่ด้านล่างของหน้าจอแชท LINE OA
ช่วยให้ผู้ใช้เข้าถึงฟังก์ชันต่างๆ ได้ง่ายและรวดเร็ว

### ประโยชน์ของ Rich Menu
- เพิ่ม Engagement ได้ถึง 142% (ข้อมูลปี 2024)
- นำทางผู้ใช้ไปยังเนื้อหาที่ต้องการ
- ลดภาระการตอบคำถามซ้ำๆ
- สร้างประสบการณ์ที่เป็นมืออาชีพ

## 4. Rich Menu Action Types

### 1. URL Action (เปิดเว็บไซต์)
```json
{
  "type": "uri",
  "uri": "https://aslanwealth.com/courses"
}
```

### 2. Message Action (ส่งข้อความ)
```json
{
  "type": "message",
  "text": "ติดต่อเรา"
}
```
*เมื่อกด จะส่งข้อความ "ติดต่อเรา" และ Chatbot ตอบกลับ*

### 3. Postback Action (ส่งข้อมูลกลับไป Server)
```json
{
  "type": "postback",
  "data": "action=contact"
}
```
*สำหรับระบบที่ซับซ้อน ใช้กับ Webhook*

## 5. Design Guidelines

### สีที่แนะนำ (Brand Colors)
```
Primary:    #1A365D (น้ำเงินเข้ม)
Secondary:  #2B6CB0 (น้ำเงิน)
Accent:     #38A169 (เขียว - เงิน/การเติบโต)
Background: #FFFFFF (ขาว)
Text:       #1A202C (ดำ)
```

### Typography
```
Font Family: Prompt, Noto Sans Thai, sans-serif
Font Size:
  - หัวข้อ: 32-40 px
  - คำอธิบาย: 24-28 px
Font Weight:
  - หัวข้อ: Bold (700)
  - คำอธิบาย: Regular (400)
```

### Icon Style
```
✅ ใช้ Icon สไตล์เดียวกันทั้งหมด
✅ ขนาด Icon: 80-120 px
✅ สี: เข้ากับ Brand Colors
✅ อาจใช้ Emoji หรือ Custom Icon
```

## 8. วิธีสร้าง Rich Menu ใน LINE OA Manager

### Step 1: เข้าสู่ LINE Official Account Manager
```
1. ไปที่ https://manager.line.biz
2. Login ด้วย LINE Account
3. เลือก LINE OA ที่ต้องการ
```

### Step 2: สร้าง Rich Menu
```
1. ไปที่ "Home" > "Rich Menus"
2. คลิก "Create"
3. ตั้งค่าพื้นฐาน:
   - Title: ชื่อเมนู (ผู้ใช้ไม่เห็น)
   - Schedule: ช่วงเวลาที่แสดง
   - Menu Bar: ข้อความบน Menu Bar
```

### Step 3: อัพโหลดรูปภาพ
```
1. คลิก "Upload Image"
2. เลือกไฟล์ (2500 x 1686 px หรือ 2500 x 843 px)
3. รอ Upload สำเร็จ
```

### Step 4: กำหนด Action Areas
```
1. เลือก Template Layout (จำนวนช่อง)
2. กำหนด Action สำหรับแต่ละช่อง:
   - Type: URL / Message / Postback
   - Content: ลิงก์ / ข้อความ / Data
```

### Step 5: บันทึกและเปิดใช้งาน
```
1. ตรวจสอบ Preview
2. คลิก "Save"
3. เปิด Toggle "Default Menu" เป็น ON
```

## 11. A/B Testing Rich Menu

### สิ่งที่ควรทดสอบ
```
1. Layout: 6 ช่อง vs 3 ช่อง
2. สี: สีเข้ม vs สีอ่อน
3. Icon Style: Emoji vs Custom Icon
4. ข้อความ: สั้น vs ยาว
5. CTA: "สมัครเลย" vs "เริ่มต้นเรียน"
```

### วิธีทดสอบ
```
1. สร้าง Rich Menu 2 แบบ
2. ใช้ Audience Segment แบ่งกลุ่ม
3. รัน 2 สัปดาห์
4. วัดผล: CTR ของแต่ละปุ่ม
5. เลือกแบบที่ดีกว่าเป็น Default
```

## 12. Best Practices

### Do's ✅
```
✅ ใช้ Icon/รูปที่ชัดเจน เข้าใจง่าย
✅ ข้อความสั้นกระชับ
✅ ใช้สีที่ตรงกับ Brand
✅ จัดวางฟังก์ชันสำคัญไว้มุมซ้ายบน
✅ อัพเดท Rich Menu ตามแคมเปญ
✅ ทดสอบบนมือถือจริง
```

### Don'ts ❌
```
❌ ใส่ข้อมูลมากเกินไป
❌ ใช้สีที่อ่านยาก
❌ ใช้ Font ขนาดเล็กเกินไป
❌ ใช้รูปที่มีความละเอียดต่ำ
❌ ละเลยการทดสอบ
```

## 13. Measurement

### เมตริกที่ควรติดตาม
```
1. Rich Menu Click Rate (ต่อปุ่ม)
2. Action Completion Rate
3. Bounce Rate (กดแล้วไม่ทำอะไรต่อ)
4. Most Popular Button
5. Least Popular Button
```

### วิธีดูสถิติ
```
LINE OA Manager > Analysis > Rich Menu
- ดู Click Rate แต่ละปุ่ม
- ดูช่วงเวลาที่มีการใช้งานมาก
```
