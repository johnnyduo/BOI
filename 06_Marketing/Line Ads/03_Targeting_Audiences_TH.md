## 4. Custom Audiences (กลุ่มเป้าหมายกำหนดเอง)

### 4.1 Website Custom Audience
สร้างจาก LINE Tag บนเว็บไซต์

| Audience | เงื่อนไข | ใช้สำหรับ |
|----------|---------|----------|
| All Visitors | เข้าชมเว็บทุกหน้า | Retargeting กว้าง |
| Course Page Viewers | ดูหน้าคอร์ส | Retarget คนสนใจคอร์ส |
| Blog Readers | อ่านบทความ >2 นาที | Retarget คนสนใจเนื้อหา |
| Cart Abandoners | ใส่ตะกร้าแต่ไม่ซื้อ | Retarget ปิดการขาย |
| Converters | สมัคร/ซื้อแล้ว | Exclude / Upsell |

### 4.2 LINE OA Custom Audience
สร้างจากผู้ติดตาม LINE OA

| Audience | เงื่อนไข | ใช้สำหรับ |
|----------|---------|----------|
| All Friends | ผู้ติดตามทั้งหมด | Exclude จาก Gain Friends |
| Active Friends | Engage 30 วันที่ผ่านมา | Upsell / Promotion |
| Inactive Friends | ไม่ Engage 30+ วัน | Re-engagement |
| Blocked Users | บล็อก OA | Retarget (ฟีเจอร์พิเศษ) |

### 4.3 Upload Custom Audience
อัพโหลดข้อมูลลูกค้าเดิม

```
รองรับการ Match:
- เบอร์โทรศัพท์
- Email
- LINE User ID
```

## 5. Lookalike Audiences

### การสร้าง Lookalike
ระบบจะหาคนที่มีลักษณะคล้ายกับ Custom Audience ที่เลือก

| Source Audience | Lookalike Size | ใช้สำหรับ |
|-----------------|----------------|----------|
| Converters (ซื้อแล้ว) | 1-3% | หาลูกค้าใหม่คุณภาพสูง |
| Website Visitors | 1-5% | ขยายกลุ่มเป้าหมาย |
| LINE OA Friends | 1-3% | หาคนคล้ายผู้ติดตาม |

### วิธีสร้าง Lookalike
1. ไปที่ LINE Ads Manager > Audiences
2. เลือก "Create Audience" > "Lookalike"
3. เลือก Source Audience
4. กำหนด % ความคล้าย (1% = คล้ายมากที่สุด)
5. บันทึกและรอระบบประมวลผล (1-2 วัน)

## 6. Audience Strategy by Campaign

### แคมเปญ 1: เพิ่มเพื่อน
```
Layer 1: Broad Targeting
- อายุ: 25-45
- พื้นที่: กรุงเทพฯ และปริมณฑล
- ความสนใจ: การเงิน, การลงทุน
- Exclude: LINE OA Friends (ติดตามแล้ว)

Layer 2: Lookalike
- Lookalike จาก Converters
- Lookalike จาก Active LINE OA Friends
```

### แคมเปญ 2: Brand Awareness
```
Broad Targeting (เข้าถึงมากที่สุด)
- อายุ: 22-55
- พื้นที่: ทั่วประเทศ (เน้นเมืองใหญ่)
- ความสนใจ: ธุรกิจ, การเงิน, พัฒนาตัวเอง
```

### แคมเปญ 3: Website Traffic
```
Layer 1: Interest-based
- อายุ: 25-45
- ความสนใจ: คอร์สออนไลน์, การลงทุน

Layer 2: Lookalike
- Lookalike จาก Website Visitors
```

### แคมเปญ 4: Retargeting
```
Layer 1: Website Retargeting
- ผู้เข้าชมเว็บ 30 วันที่ผ่านมา
- Exclude: Converters

Layer 2: LINE OA Retargeting
- Inactive Friends (30+ วัน)
- Blocked Users
```

## 8. Testing Strategy

### Audience Testing Matrix
| Test | Audience A | Audience B | งบ/กลุ่ม |
|------|------------|------------|---------|
| 1 | อายุ 25-35 | อายุ 35-45 | 5,000 บาท |
| 2 | กรุงเทพฯ only | ทั่วประเทศ | 5,000 บาท |
| 3 | Interest: หุ้น | Interest: คริปโต | 5,000 บาท |
| 4 | Lookalike 1% | Lookalike 3% | 5,000 บาท |

### วิธีวิเคราะห์ผล
หลังจากรัน 7 วัน (Learning Phase) ดู:
1. Cost per Friend (CPF)
2. CTR (Click Through Rate)
3. Conversion Rate
4. Engagement Rate

กลุ่มที่ CPF ต่ำที่สุดและ Conversion สูงสุด = ชนะ
