## 2. ข้อกำหนด LINE Ads สำหรับบริการทางการเงิน

### ประเภทธุรกิจที่ต้องได้รับอนุญาต
```
✅ ต้องมีใบอนุญาตจากหน่วยงานกำกับดูแล:
- ที่ปรึกษาการลงทุน (Investment Advisor)
- ผู้จัดการกองทุน (Fund Manager)
- นายหน้าซื้อขายหลักทรัพย์ (Securities Broker)
- ธุรกิจสินทรัพย์ดิจิทัล (Digital Asset Business)
- บริการชำระเงิน (Payment Service)
- สินเชื่อและการเงิน

❌ ไม่สามารถโฆษณาได้:
- ธุรกิจที่ไม่มีใบอนุญาต
- Binary Options
- Forex Trading ที่ไม่ได้รับอนุญาต
- Ponzi Scheme / แชร์ลูกโซ่
```

## 3. คำเตือนความเสี่ยงที่ต้องใส่

### คำเตือนมาตรฐาน (ต้องใส่ทุกชิ้นงาน)

#### สำหรับการลงทุนทั่วไป
```
"การลงทุนมีความเสี่ยง ผู้ลงทุนควรศึกษาข้อมูลก่อนการตัดสินใจลงทุน"
```

#### สำหรับหุ้นและหลักทรัพย์
```
"การลงทุนในหลักทรัพย์มีความเสี่ยง ผู้ลงทุนอาจขาดทุนจากเงินลงทุนได้
ผู้ลงทุนควรทำความเข้าใจลักษณะสินค้า เงื่อนไขผลตอบแทน และความเสี่ยงก่อนตัดสินใจลงทุน"
```

#### สำหรับกองทุนรวม
```
"การลงทุนในกองทุนรวมมีความเสี่ยง ผู้ลงทุนควรทำความเข้าใจลักษณะสินค้า
เงื่อนไข ผลตอบแทน และความเสี่ยง ก่อนตัดสินใจลงทุน"
```

#### สำหรับ Cryptocurrency
```
"สินทรัพย์ดิจิทัลมีความเสี่ยงสูง ผู้ลงทุนควรศึกษาข้อมูลและทำความเข้าใจ
ก่อนการลงทุน และลงทุนด้วยเงินที่พร้อมจะสูญเสีย"
```

### ตำแหน่งที่ต้องใส่คำเตือน
```
✅ ใส่ที่ด้านล่างของชิ้นงานโฆษณา
✅ ใช้ขนาดตัวอักษรที่อ่านได้ชัด (ไม่เล็กเกินไป)
✅ ใส่ใน Landing Page ด้วย
✅ ใส่ในข้อความ LINE OA ที่เกี่ยวกับการลงทุน
```

## 5. ข้อกำหนดเนื้อหาโฆษณา

### สิ่งที่ต้องระบุ
```
✅ ชื่อบริษัท/แบรนด์ที่ชัดเจน
✅ ข้อมูลติดต่อที่แท้จริง
✅ คำเตือนความเสี่ยง
✅ แหล่งที่มาของข้อมูล (ถ้าอ้างอิง)
✅ เงื่อนไขของข้อเสนอ (ถ้ามี)
```

### การใช้ตัวเลขและสถิติ
```
❌ "หุ้นขึ้น 500%!" (ไม่ระบุช่วงเวลา)
✅ "หุ้น XYZ เพิ่มขึ้น 50% ในรอบ 1 ปี (ข้อมูล ณ 31 ธ.ค. 2024)"

❌ "ลูกค้า 10,000 คนพอใจ" (ไม่มีหลักฐาน)
✅ "จากผู้ตอบแบบสอบถาม 500 ราย มี 85% แนะนำบริการเรา"
```

### การใช้รูปภาพ
```
✅ ใช้รูปจริงของทีมงาน/วิทยากร
✅ ใช้กราฟิกที่ไม่ทำให้เข้าใจผิด
✅ ใส่ Disclaimer หากใช้ภาพจำลอง

❌ ใช้รูปคนดัง/ดาราโดยไม่ได้รับอนุญาต
❌ ใช้รูปที่ทำให้เข้าใจผิดว่าเป็นข่าว
❌ ใช้กราฟที่บิดเบือนข้อมูล
```

## 8. กระบวนการอนุมัติโฆษณา LINE

### ขั้นตอนการอนุมัติ
```
1. ส่งโฆษณา → รอ Review 24-48 ชม.
2. ถ้าผ่าน → โฆษณาจะรันอัตโนมัติ
3. ถ้าไม่ผ่าน → จะได้รับเหตุผล → แก้ไขแล้วส่งใหม่
```

### เหตุผลที่มักถูก Reject
```
1. ไม่มีคำเตือนความเสี่ยง
2. รับประกันผลตอบแทน
3. ใช้ภาพที่มีลิขสิทธิ์
4. เนื้อหาเกินจริง
5. Landing Page ไม่ตรงกับโฆษณา
6. ไม่มีข้อมูลติดต่อ
```

### วิธีอุทธรณ์
```
1. อ่านเหตุผลที่ถูก Reject อย่างละเอียด
2. แก้ไขตามที่ระบุ
3. ส่งโฆษณาใหม่
4. ถ้ายังไม่ผ่าน ติดต่อ LINE Support
```

## 9. กฎหมายที่เกี่ยวข้อง

### พ.ร.บ. หลักทรัพย์และตลาดหลักทรัพย์
```
- ห้ามเผยแพร่ข้อมูลเท็จเกี่ยวกับหลักทรัพย์
- ห้ามชักจูงโดยใช้ข้อมูลที่ทำให้เข้าใจผิด
- ต้องได้รับใบอนุญาตในการให้คำปรึกษาการลงทุน
```

### พ.ร.บ. คุ้มครองผู้บริโภค
```
- ห้ามโฆษณาเกินจริง
- ห้ามใช้ข้อความที่ทำให้เข้าใจผิด
- ต้องแสดงข้อมูลที่เป็นประโยชน์ต่อผู้บริโภค
```

### พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล (PDPA)
```
- ต้องขอความยินยอมก่อนเก็บข้อมูลส่วนบุคคล
- ต้องแจ้งวัตถุประสงค์การใช้ข้อมูล
- ต้องมีนโยบายความเป็นส่วนตัว
```

## 10. แนวทางปฏิบัติที่ดี

### Best Practices สำหรับ Aslan Wealth Channel

#### 1. โปร่งใส
```
✅ เปิดเผยข้อมูลบริษัทอย่างชัดเจน
✅ แสดงใบอนุญาต/ใบรับรอง (ถ้ามี)
✅ ให้ข้อมูลที่ครบถ้วนและถูกต้อง
```

#### 2. ซื่อสัตย์
```
✅ ไม่พูดเกินจริง
✅ แสดงทั้งโอกาสและความเสี่ยง
✅ ใช้ข้อมูลจริง มีแหล่งอ้างอิง
```

#### 3. รับผิดชอบ
```
✅ ให้ความรู้ที่ถูกต้อง
✅ ไม่ชักจูงให้ลงทุนเกินตัว
✅ มีช่องทางให้ร้องเรียน
```
