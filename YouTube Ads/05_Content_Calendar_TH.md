## 6. Video Content Templates

### Long-form Video Structure (8-15 นาที)

```
0:00-0:30   Hook (ทำไมต้องดู?)
0:30-1:00   Intro + Agenda
1:00-3:00   Point 1 (พร้อมตัวอย่าง)
3:00-5:00   Point 2 (พร้อมตัวอย่าง)
5:00-7:00   Point 3 (พร้อมตัวอย่าง)
7:00-8:00   Summary + Key Takeaways
8:00-8:30   CTA: Subscribe, Like, Comment
8:30-9:00   Outro + End Screen
```

### Shorts Structure (30-60 วินาที)

```
0:00-0:01   Hook (Attention grab)
0:01-0:05   Setup (บอกว่าจะได้อะไร)
0:05-0:45   Content (เนื้อหาหลัก)
0:45-0:55   Key Takeaway
0:55-1:00   CTA: Follow, See more
```

### Ad Video Structure (30 วินาที)

```
0:00-0:03   Hook (ก่อน Skip ได้)
0:03-0:05   Problem (ปัญหาที่คนมี)
0:05-0:15   Solution (Aslan แก้ได้ยังไง)
0:15-0:25   Benefits (ได้อะไรบ้าง)
0:25-0:30   CTA (Subscribe/Visit)
```

## 9. Content Pillars

### 4 Pillars หลัก

```
1. Education (40%)
   - วิธีลงทุน
   - ความรู้พื้นฐาน
   - อธิบายศัพท์การเงิน

2. Protection (25%)
   - ป้องกัน Scam
   - รู้เท่าทันมิจฉาชีพ
   - ความปลอดภัยทางการเงิน

3. Inspiration (20%)
   - เรื่องราวความสำเร็จ
   - Motivation
   - เป้าหมายการเงิน

4. Entertainment (15%)
   - Trends
   - ข่าวสาร
   - Memes/Relatable content
```

## 10. Posting Checklist

### ก่อนโพสต์

- [ ] Title มี keyword และน่าคลิก
- [ ] Thumbnail โดดเด่น ชัดเจน
- [ ] Description 200+ คำ มี keyword
- [ ] Tags 10-15 tags
- [ ] Cards ใส่แล้ว (2-5 ตัว)
- [ ] End Screen ใส่ Subscribe + วิดีโอแนะนำ
- [ ] Hashtags 3 อัน
- [ ] Schedule เวลา 19:00 (หรือ 10:00 สำหรับ สส-อา)

### หลังโพสต์ (ใน 1 ชั่วโมงแรก)

- [ ] Pin Comment ใส่ CTA
- [ ] โพสต์ Community Tab
- [ ] แชร์ไป LINE OA
- [ ] แชร์ไป X (Twitter)
- [ ] ตอบ Comment แรกๆ
- [ ] เช็ค Real-time Analytics
