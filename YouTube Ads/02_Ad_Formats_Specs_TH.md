## 1. รูปแบบโฆษณา YouTube ทั้งหมด

### 1.1 Skippable In-stream Ads

```
┌─────────────────────────────────────────────────┐
│  📺 วิดีโอโฆษณา                                   │
│                                                 │
│  [=====================================]  0:15  │
│  ▶️                                              │
│                                                 │
│  ⏭️ Skip Ad (หลัง 5 วินาที)                      │
└─────────────────────────────────────────────────┘
```

| รายการ | สเปค |
|--------|------|
| ความยาว | ไม่จำกัด (แนะนำ 15-60 วินาที) |
| ข้ามได้หลัง | 5 วินาที |
| Resolution | 1920x1080 หรือ 1280x720 |
| Aspect Ratio | 16:9 |
| Format | MP4, MOV, AVI |
| File Size | Max 1GB |
| Billing | CPV (ดูครบ 30 วิ หรือ คลิก) |
| CPV เฉลี่ย (ไทย) | 0.30-0.80 บาท |

**เหมาะสำหรับ:**
- Brand Awareness
- Video Views
- Product launches
- Storytelling

**Best Practices:**
- Hook ใน 5 วินาทีแรก (ก่อนข้ามได้)
- ใส่ Logo ตั้งแต่ต้น
- CTA ชัดเจน
- คุณภาพเสียงดี (หลายคนฟังมากกว่าดู)

## 2. Video Specs รวม

### Resolution & Quality

| Resolution | ชื่อเรียก | แนะนำ |
|------------|----------|-------|
| 3840x2160 | 4K UHD | Premium content |
| 1920x1080 | Full HD | ✅ แนะนำ |
| 1280x720 | HD | ยอมรับได้ |
| 854x480 | SD | ไม่แนะนำ |

### Video Format

| Format | Extension | แนะนำ |
|--------|-----------|-------|
| H.264 MP4 | .mp4 | ✅ ดีที่สุด |

| WMV | .wmv | ไม่แนะนำ |

### Audio Specs

| รายการ | สเปค |
|--------|------|
| Codec | AAC-LC |
| Bitrate | 128 kbps+ (แนะนำ 256) |
| Sample Rate | 48 kHz |
| Channels | Stereo |

### File Size

| ประเภท | Max Size |
|--------|----------|
| In-stream | 1 GB |
| Shorts | 500 MB |
| General | แนะนำ < 500 MB |

## 3. Thumbnail Specs

```
┌─────────────────────────────────────┐
│                                     │
│         Thumbnail Image             │
│         1280 x 720 px               │
│         (16:9)                      │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  TEXT ไม่เกิน                │   │
│  │  20% ของภาพ                 │   │
│  └─────────────────────────────┘   │
│                                     │
└─────────────────────────────────────┘
```

| รายการ | สเปค |
|--------|------|
| Resolution | 1280x720 px |
| Aspect Ratio | 16:9 |
| Format | JPG, PNG, GIF |
| File Size | Max 2 MB |
| Text | ไม่เกิน 20% |

### Thumbnail Best Practices

**Do's:**
- ใบหน้าคน + อารมณ์ชัดเจน
- ตัวเลขใหญ่ (3, 5, 10)
- สีตัดกันกับพื้นหลัง YouTube (แดง)
- ความละเอียดสูง
- Text อ่านง่ายบนมือถือ

**Don'ts:**
- ภาพเบลอ
- Text เยอะเกินไป
- สีน้ำเงิน/ขาวล้วน (กลืนกับ YouTube)
- Clickbait ที่ไม่ตรงเนื้อหา

## 5. ตัวอย่างโครงสร้างวิดีโอโฆษณา

### 15-second Non-skippable

```
0-3 sec:   Hook + Problem
3-10 sec:  Solution + Benefits
10-14 sec: CTA + Offer
14-15 sec: Logo + End
```

### 30-second Skippable

```
0-5 sec:   Strong Hook (ก่อนข้ามได้)
5-15 sec:  Problem + Empathy
15-25 sec: Solution + Benefits
25-28 sec: CTA
28-30 sec: Logo + End
```

### 6-second Bumper

```
0-1 sec: Attention grab
1-4 sec: Key message (1 อย่างเท่านั้น)
4-6 sec: Logo + CTA
```

### 60-second Shorts

```
0-1 sec:   Hook (ดึงความสนใจทันที)
1-10 sec:  Setup/Problem
10-40 sec: Content/Solution
40-55 sec: Key takeaway
55-60 sec: CTA + Follow
```

## 6. Creative Checklist

### Before Upload

- [ ] Resolution: 1080p หรือสูงกว่า
- [ ] Format: MP4 (H.264)
- [ ] Audio: Clear, 128kbps+
- [ ] File Size: < 500MB
- [ ] Aspect Ratio: ถูกต้อง (16:9 หรือ 9:16)
- [ ] Length: เหมาะกับ format

### Content Check

- [ ] Hook ใน 5 วินาทีแรก
- [ ] CTA ชัดเจน
- [ ] Logo แสดงอย่างน้อย 3 วินาที
- [ ] ไม่มี copyright music/content
- [ ] ถูกกฎ YouTube Ads Policy
- [ ] คำเตือนการลงทุน (ถ้าจำเป็น)

### Thumbnail Check

- [ ] 1280x720 px
- [ ] Text อ่านได้บนมือถือ
- [ ] สีโดดเด่น
- [ ] ไม่ Clickbait เกินไป
- [ ] ตรงกับเนื้อหาวิดีโอ
