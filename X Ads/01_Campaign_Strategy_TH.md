## 3. รูปแบบโฆษณา X Ads

### 3.1 Promoted Posts (แนะนำ!)
```
โพสต์ปกติที่ Boost ให้เข้าถึงคนมากขึ้น
- Text + Image/Video
- Engagement objective
- CPE: 2-8 บาท
```

### 3.2 Follower Ads
```
โฆษณาเพื่อเพิ่มผู้ติดตาม
- แสดงใน "Who to follow"
- Cost per Follow: 10-30 บาท
```

### 3.3 Website Cards
```
โพสต์ที่มี Link Preview ใหญ่
- Headline + Image + URL
- CPC: 3-10 บาท
```

### 3.4 Video Ads
```
วิดีโอ Auto-play ใน Feed
- Max 2:20 นาที (Free) / 60 นาที (Premium)
- CPV: 1-3 บาท (ดู 6 วินาที)
```

### 3.5 Carousel Ads
```
หลายภาพ/วิดีโอ Swipe ได้
- 2-6 cards
- เหมาะกับ Storytelling
```
