## 1. Single Tweet Templates

### Template: Quick Tip
```
💡 เคล็ดลับการลงทุน:

[Tip สั้นๆ 1-2 ประโยค]

Save ไว้อ่านอีก 🔖
```

### Template: Question/Poll
```
คำถาม: [คำถามที่กระตุ้นให้ตอบ]

🔘 ตัวเลือก A
🔘 ตัวเลือก B
🔘 ตัวเลือก C
🔘 ตัวเลือก D

บอกเหตุผลใน Reply ด้วยนะ 👇
```

### Template: Hot Take
```
Unpopular opinion:

[ความเห็นที่ขัดแย้งกับคนส่วนใหญ่]

เห็นด้วยไหม? 🤔
```

### Template: News Commentary
```
📰 [ข่าว/เหตุการณ์]

ความเห็นผม:
[วิเคราะห์สั้นๆ]

คิดยังไงกันบ้าง?
```

### Template: Stat/Fact
```
🔢 ตัวเลขที่น่าตกใจ:

[สถิติ/ข้อเท็จจริง]

นี่คือสิ่งที่คุณต้องรู้ 👇
```

## 2. Thread Templates

### Thread: Educational (10 tweets)
```
1/ [HOOK - ปัญหา/คำถาม]

ผมศึกษาเรื่อง [หัวข้อ] มา [เวลา]

นี่คือ [จำนวน] บทเรียนที่สำคัญที่สุด

🧵 Thread:

2/ บทเรียนที่ 1: [หัวข้อ]

[อธิบาย 2-3 ประโยค]

3/ บทเรียนที่ 2: [หัวข้อ]

[อธิบาย 2-3 ประโยค]

...

10/ สรุป:

✅ [Point 1]
✅ [Point 2]
✅ [Point 3]

ถ้าชอบ Thread นี้:
• Like Tweet แรก
• Repost ให้เพื่อน
• Follow @AslanWealth

[Reply ถัดไป: ใส่ลิงก์ถ้ามี]
```

### Thread: Story Format
```
1/ [HOOK - เหตุการณ์ที่น่าสนใจ]

เมื่อ [เวลา] ที่แล้ว [เหตุการณ์]

นี่คือเรื่องราวทั้งหมด 👇

2/ มันเริ่มจาก...

[เล่าเรื่อง]

3/ แล้ว...

[เล่าต่อ]

...

8/ บทเรียนที่ได้:

[บทเรียน]

9/ ถ้าผมย้อนเวลาได้ ผมจะบอกตัวเองว่า:

[คำแนะนำ]

10/ หวังว่าเรื่องนี้จะเป็นประโยชน์

Follow @AslanWealth สำหรับเนื้อหาแบบนี้ทุกวัน 🦁
```
