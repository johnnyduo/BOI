## 1. รูปแบบโฆษณา X ทั้งหมด

| รายการ | สเปค |
|--------|------|
| ขนาดภาพ | 1200 x 675 px (16:9) หรือ 1200 x 1200 px (1:1) |

| File Size | Max 5 MB (ภาพ), 15 MB (GIF) |
| Text | 280 ตัวอักษร (Free) / 25,000 (Premium) |

| รายการ | สเปค |

| File Size | Max 512 MB (Free) / 8 GB (Premium) |
