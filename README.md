## Project Description

Physical AI Agent + Multi-Agent AI Platform for Financial Literacy and Protection for Thai citizens.

**Key Products:**
1. **Physical AI Agent (Aslan Mascot)** - AI-powered robot for financial education
2. **Thai NLP Engine** - Natural language processing for Thai scam detection
3. **Multi-Agent AI Platform** - 5 coordinated AI agents for financial analysis
4. **Explainable AI Module** - Transparent AI decision explanations in Thai
5. **Mobile Application** - Connected app for monitoring and learning

## Submission Workflow

### Step 1: Prepare Documents
Use files in `01_Submission/`:
- Copy data from `BOI_COMPLETE_CONSOLIDATED.docx` to BOI online form
- Refer to `BOI_SUBMISSION_COMPLETE.md` for complete data reference

### Step 2: Print Attachments
Print as PDF from `02_Attachments/`:
- `attachment_a_rd_COMPLETE.html` → Attachment A (R&D)
- `attachment_c_digital_COMPLETE.html` → Attachment C (Digital)
- `attachment_d_product_details.html` → Attachment D (Products)

### Step 3: Include Government Documents
Attach files from `05_Government_Documents/`:
- Memorandum of Association
- Electronic Certificate
- NDA Agreement

### Step 4: Submit via BOI Online
Submit complete package through BOI e-Investment system.

## Google Apps Scripts Usage

Located in `03_Scripts/`:

| Script | Purpose |
|--------|---------|
| `BOI_Form_Generator.gs` | Auto-generates BOI form data in Google Sheets |
| `BOI_Presentation_Generator.gs` | Creates 10-slide BOI project presentation |
| `BOI_Workflow_Slides.gs` | Generates 13-slide technical workflow visualization |

**To use:**
1. Open Google Slides and create a new presentation
2. Go to Extensions > Apps Script
3. Paste the script content
4. Run the main function (e.g., `createBOIWorkflowPresentation()`)
5. Grant permissions when prompted
