## 1. หลักการเขียน Ad Copy

### สูตร AIDA
```
A - Attention: ดึงดูดความสนใจ
I - Interest: สร้างความสนใจ
D - Desire: กระตุ้นความต้องการ
A - Action: เรียกร้องให้ทำ
```

### สูตร PAS
```
P - Problem: ระบุปัญหา
A - Agitate: ขยายปัญหา
S - Solution: นำเสนอทางออก
```

## 9. A/B Test Checklist

### สิ่งที่ควรทดสอบ
```
□ Headline: ความรู้ฟรี vs Pain Point
□ Tone: เป็นทางการ vs กันเอง
□ Length: สั้น vs ยาว
□ Emoji: มี vs ไม่มี
□ Offer: ส่วนลด vs ของแถม
□ Urgency: มี Deadline vs ไม่มี
□ CTA: "แอดเพื่อน" vs "เริ่มเลย"
```

## 10. Writing Checklist

### ก่อนเผยแพร่
```
□ ตรวจสอบตัวสะกด
□ นับจำนวนตัวอักษร (ไม่เกินกำหนด)
□ มีคำเตือนความเสี่ยง
□ ไม่มีการรับประกันผลตอบแทน
□ CTA ชัดเจน
□ ทดสอบอ่านบนมือถือ
□ ตรงกับ Creative/Visual
```
